# Cloud Infrastructure Operations & DevOps Foundations

## Overview
This project demonstrates foundational knowledge of cloud infrastructure operations and DevOps practices aligned with entry-level cloud and DevOps roles.

## Architecture
The project is based on an Azure-aligned cloud architecture consisting of a user, resource group, virtual network, virtual machine, and monitoring components.

## Key Areas Covered
- Cloud infrastructure design and hierarchy
- Monitoring and alerting concepts
- Role-Based Access Control (RBAC)
- Governance best practices
- DevOps fundamentals using Git and CI pipelines

## Tools and Technologies
- Git and GitHub
- GitHub Actions (CI/CD)
- Architecture design tools
- Markdown documentation

## Learning Outcomes
Developed an understanding of cloud operations, security principles, DevOps workflows, and enterprise-style documentation practices.
