# Monitoring and Operations

## Metrics Monitored
- CPU Utilization
- Memory Usage
- Disk Usage
- Virtual Machine Availability

## Alert Conditions
- CPU utilization exceeds 80% for 5 minutes
- Virtual Machine is stopped or unresponsive

## Daily Health Check Checklist
- Verify VM running status
- Review monitoring dashboards
- Check alerts and logs
- Report issues to senior team members

## Incident Handling (Basic)
- Identify the issue from alerts
- Perform initial checks
- Document the issue
- Escalate if required
