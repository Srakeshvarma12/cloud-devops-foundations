# Role-Based Access Control (RBAC)

## Roles Defined
- Admin: Full access to manage all cloud resources
- Contributor: Can create and modify resources
- Reader: Read-only access to view resources

# Governance Policies

## Policy Rules
- Restrict resource creation to approved regions
- Enforce standard naming conventions
- Prevent unauthorized deletion of critical resources

## Purpose
These controls help maintain security, compliance, and operational consistency in cloud environments.
